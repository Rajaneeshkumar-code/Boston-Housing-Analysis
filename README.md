# Boston-Housing-Analysis
📌 Overview

This project focuses on analyzing the Boston Housing dataset to understand the factors affecting house prices and building predictive models. It was developed as part of applying concepts learned from Statistics for Data Science with Python.
🎯 Objectives
Perform data exploration and cleaning
Visualize relationships between housing features
Apply statistical techniques (Hypothesis Testing)
Build regression models for price prediction
Evaluate model performance
📂 Project Files
Boston-Housing-Analysis/
│── FinalProject_Boston Housing.ipynb   # Main project notebook
│── SUMMARY_OF_STAT.pdf                # Statistical concepts summary
│── README.md                          # Project documentation
📊 Project Workflow
🔍 1. Data Exploration
Loaded dataset and checked structure
Identified missing values and feature types
📈 2. Exploratory Data Analysis (EDA)
Used visualizations to understand patterns
Analyzed correlations between variables
📉 3. Statistical Analysis
Applied hypothesis testing
Interpreted statistical significance of features
🤖 4. Model Building
Implemented regression models
Compared different approaches
📏 5. Evaluation
Used metrics like:
Mean Squared Error (MSE)
Root Mean Squared Error (RMSE)
R² Score
