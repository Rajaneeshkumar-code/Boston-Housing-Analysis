> A comprehensive data science project analyzing the Boston Housing dataset to identify key factors affecting house prices and build predictive regression models.

---

## 📌 Overview

This project explores the **Boston Housing dataset** through a full data science workflow — from exploratory analysis and hypothesis testing to building and evaluating machine learning regression models. It was developed as part of applying concepts learned from **Statistics for Data Science with Python**.

---

## 🎯 Objectives

- 🔍 Perform data exploration and cleaning
- 📊 Visualize relationships between housing features
- 🧪 Apply statistical techniques (Hypothesis Testing)
- 🤖 Build regression models for price prediction
- 📈 Evaluate model performance using standard metrics

## 🔄 Project Workflow

### 1. 🔎 Data Exploration
- Loaded dataset and checked structure
- Identified missing values and feature types

### 2. 📊 Exploratory Data Analysis (EDA)
- Used visualizations to understand patterns
- Analyzed correlations between variables

### 3. 📐 Statistical Analysis
- Applied hypothesis testing
- Interpreted statistical significance of features

### 4. 🏗️ Model Building
- Implemented regression models
- Compared different modeling approaches

### 5. ✅ Evaluation
Used the following metrics to assess model performance:
- **MSE** — Mean Squared Error
- **RMSE** — Root Mean Squared Error
- **R² Score** — Coefficient of Determination
